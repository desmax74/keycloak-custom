Dell Software Developers Platform use OAuth 2.0 for authorization. Follow the steps below to get authorized.

1. Get an API Account

Register for your Dell Software Developers account.

2. Request Access Token

Access token is unique to a user and their role. Some services requires OAuth2 authentication, in these cases, the headers must include a valid authentication token.

Example HTTP headers: Accept: application/json Authorization: Bearer {access token}

3. Make the call

Once you have successfully obtained an access token, you can start making authenticated API requests.

See examples for more information.
